# Retatrutide Supplier Database

Interactive dashboard of Retatrutide API suppliers — tracking manufacturers, brokers, pricing, documentation, and outreach status.

**Navigate:** open `index.html` in any browser, or view on GitHub Pages.

## Data sources
- Janoshik public test registry (53 Retatrutide tests, scraped Jul 2026)
- Live web research on manufacturer and contact-form pages
- Direct supplier inquiry responses
- Chinese-language B2B platform and manufacturer website research

## Structure
- `index.html` — self-contained interactive database (all CSS/JS inline)
- `base_data.json` — original Janoshik test data
- `merged_data.json` — canonical machine-readable dataset
- `handoff.md` — project handoff and outreach tracking
